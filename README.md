# 🌐 Simple API with Flask

This project is a simple API built with Flask to demonstrate basic HTTP routes and JSON responses.

---

## 🚀 Features
- Creates HTTP routes
- Returns JSON responses
- Demonstrates basic API structure using Flask

---

## 🛠 Technologies
- Python
- Flask

---

## ▶️ How to run

1. Install dependencies:
```bash
pip install flask
```

2. Run the application:
```bash
python app.py
```

3. Open in your browser:
```bash
http://127.0.0.1:5000/
```

4. Test the API endpoint:
```bash
http://127.0.0.1:5000/api/data
```

---

## 📂 Project Structure
- app.py → main application file

---

## 🎯 Purpose
This project demonstrates basic API development with Flask, including route creation and JSON response handling.
